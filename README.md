Менеджер задач — ToDo

Использоване при разработке технологии:

◉ Java Servlet
◉ Hibernate
◉ PostgreSQL
◉ JDBC
◉ Apache Tomcat
◉ JSP/JSTL
◉ Maven

На главной странице пользователя встречает приветственная страница, на которой ему предлагается зарегистрироваться или войти под своей учетной записью.
